TECHNOLOGY TOOLS WE AIM TO MASTER

  1. HTML

    2. HTML stands for HyperText Markup Language. It is the basic structure of the website. HTML is absolutely needed for web development because this language makes sure that all the displayed information such as images, text, or videos are properly formatted.

    Practical uses of HTML
    Click to see uses
    • Browse the Internet with ease
    • Creating web documents
    • Game development
  2. CSS

    3. CSS stands for Cascading Style Sheets. It is used to style a website. CSS is primarily used to change the visual appearance of a website by altering or adding different elements such as fonts, colors, shapes, page layouts, and element positioning based on the front-end developers aesthetic or visual design choice.

    Practical uses of CSS
    Click to see uses
    • Styling the website
    • Formatting page layouts
    • Adding animations or effects
  3. Javascript

    4. Javascript is an object-oriented programming language that adds interactivity to a web page and makes it dynamic. While HTML provides the bones and CSS provides the skin for a website, Javascript acts as the brain that adds interactivity, responsiveness, and makes it dynamic. This programming language allows the developer to implement complex features, as well as allowing users to perform complex actions in a website.

    Practical uses of Javascript
    Click to see uses
    • Combining Javascript with HTML5 in game development
    • Mobile App Development
    • Server applications
  4. Databases

    5. Databases are sorting systems used to store user data. Their role in websites is to add interactivity and save any inputs by the users such as audio, emails, video, files, etc. MySQL is a common database used for websites. MySQL is a relational database management system created by Oracle that is widely used for its reliability, performance, and ease of use.
